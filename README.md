# PS-CRM AI Sentiment Analysis

AI-powered Public Service CRM to track citizen complaints and analyze public sentiment in real time.

---

## Features
- Centralized complaint management
- AI-based sentiment analysis
- Real-time dashboard
- Booth-level sentiment insights
- Issue trend detection

---

## Tech Stack
- Python
- NLP (Transformers)
- Flask API
- Streamlit Dashboard
- Pandas / Scikit-learn
