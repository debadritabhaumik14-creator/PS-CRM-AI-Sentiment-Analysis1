# Smart Public Service CRM (PS-CRM) with AI Sentiment Analysis

## Overview
This project builds a centralized Public Service CRM system combined with an AI-powered sentiment analysis engine to monitor citizen grievances and public sentiment in real time.

The system collects complaints and social media data, analyzes sentiment, identifies key issues, and visualizes insights through dashboards and heatmaps.

## Features
- Centralized complaint management
- AI-based sentiment analysis
- Multi-language data analysis
- Booth-level and constituency-level insights
- Real-time dashboards
- Issue trend detection

## Tech Stack
- Python
- NLP (Transformers / HuggingFace)
- Flask API
- React / Streamlit Dashboard
- MongoDB / PostgreSQL
